# Architecture Decision Records

Registro de decisiones técnicas y de diseño del proyecto ORION.

Cada ADR documenta una decisión significativa: qué se decidió, por qué, y qué implicaciones tiene. Una vez aceptado, un ADR no se modifica — si la decisión cambia, se crea un nuevo ADR que lo supersede.

| [ADR-003](ADR-003-semantic-evolutionary-classifier.md) | Clasificador semántico y auto-evolutivo | Aceptado |

## Estados posibles

- **Propuesto** — en discusión, aún no adoptado
- **Aceptado** — decisión tomada y en vigor
- **Supersedido** — reemplazado por otro ADR (se indica cuál)
- **Obsoleto** — ya no aplica al estado actual del proyecto
